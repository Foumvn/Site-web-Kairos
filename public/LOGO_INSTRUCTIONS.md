# Instructions pour ajouter le logo Kairos

## Emplacement du logo

Placez votre fichier logo dans le dossier `public/` avec le nom `logo.png`

Le logo doit contenir :
- Un globe stylisé à gauche
- Le texte "KAIROS" à droite
- Le tagline "Relever les défis, c'est notre valeur !" en dessous

## Format recommandé

- **Format** : PNG avec transparence (ou SVG)
- **Nom du fichier** : `logo.png`
- **Dimensions** : Recommandé 240x120px ou plus (ratio 2:1)
- **Couleur** : Le logo s'adapte automatiquement au thème (dark/light)

## Emplacement du fichier

```
kairos-next/
  └── public/
      └── logo.png  ← Placez votre logo ici
```

Une fois le fichier `logo.png` ajouté dans `public/`, le logo apparaîtra automatiquement dans le header et le footer.

## Fallback

Si le logo n'est pas trouvé, un fallback avec l'icône "K" et le texte sera utilisé automatiquement.
